Mongockup.

This tool allows you, to manage your own backups of mongodb with a very simple UI without touch the console at all.

Manager / Features

  • Schedule Backups
  • Level of Backup [ Database, Collection, Document ]
  • Dropbox Backup Upload
  • Amazon S3 Compatibility
